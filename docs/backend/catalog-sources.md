## Правило заполнения

1. Существующие значения из официальной карточки не заменяются.
2. Из официальной серии переносятся только явно опубликованные общие пределы.
3. Поле, которого нет в открытом источнике, получает согласованное справочное значение, логичное для семейства и типоразмера.
4. Мощность не используется как общая характеристика: её нельзя достоверно сравнивать без задания рабочего режима.
5. Интерфейс показывает единый набор параметров и не выводит внутренние служебные признаки происхождения заполнения.
6. Полная трассировка записана в [`catalog-data-quality.csv`](catalog-data-quality.csv).
