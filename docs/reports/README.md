# Отчёты

Итоговый отчёт по учебной практике:

```text
docs/reports/Отчёт_по_учебной_практике_ThermoSelect.docx
docs/reports/Отчёт_по_учебной_практике_ThermoSelect.pdf
```

Оформление основано на новых материалах из папки `Материалы/`: формальная титульная страница СПбГТИ(ТУ), Times New Roman 14 pt, поля 30/15/20/20 мм, межстрочный интервал 1,5, нумерованные разделы, подписи таблиц и рисунков, схемы данных и алгоритма.

Отчёт воспроизводится из Markdown-источника и диаграмм:

```powershell
$python = 'C:\Users\PC\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe'
& $python .\tools\reports\generate-practice-report.py
```

ФИО, группа и подписи оставлены как заполняемые поля, поскольку персональные данные студента не были переданы.
