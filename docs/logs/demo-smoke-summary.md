# Проверка локального запуска

Дата проверки: 16.07.2026.

Проверен запуск собранного приложения командой:

```powershell
cd .\Авторизация
.\run-demo.ps1 -SkipBuild
```

Результаты:

- активирован профиль `demo`;
- Spring Boot запущен на `http://localhost:8080`;
- Flyway подтвердил актуальность схемы H2;
- вход `demo / demo12345` вернул роль `USER`;
- защищённый поиск вернул 50 из 50 записей;
- фильтр производителя вернул четыре модели «Ридан»;
- выгрузка того же фильтра сформировала валидную Excel-книгу из четырёх строк;
- `excludedUnknownCount` равен 0, поскольку общие параметры карточек заполнены;
- прямой маршрут `/catalog` вернул собранный React SPA с HTTP 200;
- в каталоге и карточке отсутствуют проценты совпадения, технические метки и строковые значения `null`;
- размеры отображаются целыми миллиметрами, остальные числа — максимум с одним знаком после запятой;
- исполняемый JAR успешно сформирован;
- runtime-журнал создан в `Авторизация/logs/thermoselect.log`.

Runtime-журналы и файловая база не коммитятся. Это предотвращает публикацию локальных данных и постоянно изменяющихся файлов. Правила просмотра и ротации описаны в `docs/logs/README.md`.
