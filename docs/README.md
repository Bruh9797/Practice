# Документация ThermoSelect

Документы разделены по назначению, чтобы материалы разработки и защиты не смешивались с исходным кодом.

| Каталог | Содержание |
|---|---|
| [`frontend/`](frontend/README.md) | React, маршруты, API-клиент, запуск Vite и frontend-тесты |
| [`backend/`](backend/README.md) | Spring Boot, БД, каталог, источники и алгоритм подбора |
| [`auth/`](auth/README.md) | Session-аутентификация, CSRF, роли и матрица API |
| [`tests/`](tests/README.md) | тест-кейсы, команды запуска и сохраняемые результаты |
| [`logs/`](logs/README.md) | расположение и просмотр журналов приложения и сборки |
| [`reports/`](reports/README.md) | отчёт по практике и инструкция его воспроизведения |
| [`presentation/`](presentation/README.md) | презентация к защите, реальные экраны и материалы шаблона |
| [`defense/`](defense/DEFENSE.md) | сценарий демонстрации, тезисы и ответы на вопросы |
| [`diagrams/`](diagrams/) | Mermaid/PlantUML-исходники и SVG-схемы |

Исходные материалы кафедры находятся в пользовательской папке `Материалы/` и не изменяются генераторами проекта.
