# Презентация к защите

Итоговый файл:

```text
docs/presentation/Презентация_к_защите_ThermoSelect.pptx
```

Презентация создана на основе нового образца из папки `Материалы/` и содержит 19 слайдов: постановку задачи, модели, архитектуру, алгоритм, обоснование стека, реальные экраны приложения, тестирование и выводы.

Персональные данные на титульном слайде оставлены заполняемыми. Иллюстрации интерфейса воспроизводятся локально скриптом:

```powershell
node .\tools\presentation\capture-ui-screens.mjs .\docs\presentation\assets
```
