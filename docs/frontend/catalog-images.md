# Фотографии теплообменников

Карточки каталога, детальная страница и сравнение используют локальные изображения из
`Авторизация/frontend/public/catalog-images/`. Это исключает зависимость интерфейса от внешних CDN.

Для точных конфигураций Alfa Laval используются фотографии соответствующих T2, T5 и T6. Для Danfoss,
Kelvion, SWEP, Basco, Ридан и воздушных аппаратов применяются фотографии модели либо официальной серии.
Если производитель публикует одну фотографию для диапазона конфигураций, она повторяется для записей
этой серии. Для новых записей администратора предусмотрена фотография соответствующего конструктивного
семейства; при ошибке загрузки остаётся прежняя схематическая иллюстрация.

Основные источники изображений:

- Alfa Laval — официальный магазин и раздел спиральных теплообменников;
- Danfoss Design Center — изображение B3;
- Kelvion — официальные страницы пластинчатых и воздушных серий;
- SWEP — фотография B15T, опубликованная профильным поставщиком;
- API Heat Transfer — официальные страницы Basco Type 500, U-Tube, OP и Hub;
- Ридан — официальный архив каталога разборных пластинчатых аппаратов;
- российские кожухотрубные аппараты — фотография аналогичного серийного исполнения российского изготовителя.
